La Famille Tattoo - Landing Page Concept

Este projeto foi desenvolvido como uma solução freelance para modernizar a presença digital de um estúdio de tatuagem local. O objetivo era criar um funil de vendas automatizado via WhatsApp.

Tecnologias: React, Tailwind CSS, Vite, Lucide React.

Desafios Superados:

Implementação de formulário que gera links dinâmicos para WhatsApp.

Otimização de SEO Local e Tags Open Graph.

Layout Mobile-First responsivo.
